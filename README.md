Clone of tetris but 5blocks for piece.

### Installing
A step by step series  that tell you how to get a execute project.
Get it from GitHub
```
git clone git@github.com:Przemekkkth/Pentomino_Qt-Cpp.git
```
Compile
```
qmake && make
```
